# 大大阿卢的拼音修炼手册

“大大阿卢的拼音修炼手册”是一个面向成年人的中文拼音输入学习 Web App。它帮助目前习惯笔画输入的人，逐步建立“看到汉字 -> 想到读音 -> 拆成拼音 -> 键盘输入 -> 选择正确汉字”的能力。

第一版不需要注册登录，不接数据库，不使用外部 API。学习进度、打卡、奖励、错题和经验值都保存在当前浏览器的 `localStorage` 中。

## 如何安装

```bash
npm install
```

## 如何启动

开发模式：

```bash
npm run dev
```

然后在浏览器打开命令行显示的本地地址，通常是：

```text
http://localhost:3000
```

生产构建检查：

```bash
npm run build
```

类型检查：

```bash
npm run typecheck
```

基础烟测：

```bash
npm test
```

## 如何修改题库

题库集中在 `src/data/questions.ts`。

每道题至少包含：

- `id`：题目唯一标识，不要重复。
- `type`：题型，支持 `char`、`word`、`choice`、`chat`。
- `chinese`：展示给用户的中文内容。
- `answer`：标准拼音答案，使用小写、无声调、不中断拼音。
- `difficulty`：难度。
- `lessonId`：所属知识点，对应 `src/data/lessons.ts` 中的课程 id。
- `hints`：答错后的分步提示。
- `explanation`：易错解释或答案说明。

每日任务配置在 `src/data/lessons.ts` 的 `dailyPlans`。每一天包含 1 个知识点、5 道单字题、5 道词语题和 1 道聊天题。

修改后建议运行：

```bash
npm test
npm run typecheck
```

## 如何修改奖励规则

奖励规则集中在 `src/data/config.ts` 的 `rewardRules`：

- `dailyComplete`：完成今日全部任务。
- `streak3`：连续打卡 3 天。
- `streak7`：连续打卡 7 天。
- `perfectPracticeGroup`：一组练习全部答对。

奖励名称也在同一文件中，当前完整名称为：

```text

```

## 如何修改经验和等级

经验规则在 `src/data/config.ts` 的 `xpRules`。

等级规则在 `levelRules`，当前包含：

1. 拼音萌新
2. 拼音学徒
3. 拼音进阶者
4. 拼音输入能手
5. 拼音大师

只要调整每个等级的 `minXp`，首页和“我的”页面会自动更新等级进度。

## 如何重置数据

在 App 的“我的”页面点击“重置学习数据”，再点击二次确认按钮即可清空当前浏览器内的学习数据。

也可以在浏览器开发者工具中清除本站点的 `localStorage`。项目使用的 key 是：

```text
dada-alu-pinyin-progress-v1
```

## 当前版本功能

- 固定账号：大大阿卢。
- 六个模块：今日、学习、练习、打卡、奖励、我的。
- 今日任务：知识点、5 道单字、5 道词语、1 组聊天短句。
- 四种练习：汉字转拼音、词语拼音输入、选择正确拼音、模拟聊天输入。
- 答案判断：忽略大小写；词语和聊天题自动删除空格。
- 错题提示：先提示首字母，再提示拆分，最后显示完整答案。
- 打卡：完成当天全部任务后自动点亮，同一天不会重复增加。
- 奖励：完成今日任务、连续打卡、一组全对都会记录奖励，刷新不会重复发放。
- 本地保存：刷新页面后进度保留。
- 移动端和桌面端自适应。

## 当前版本限制

- 不是真正的中文输入法引擎，只比较用户输入的预设无声调拼音。
- 不支持多账号和云同步。
- 题库仍是第一版样例数据，需要后续根据真实学习情况继续扩充。
- 没有语音播放和发音识别。
- 没有后台管理界面，修改题库需要编辑本地 TypeScript 文件。
